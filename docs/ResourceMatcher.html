ResourceMatcher

ResourceMatcher

Resource checker class that checks that any regular expressions that matches in the source also appears in the translation.

Constructor

new ResourceMatcher(options)

Description:
  • Construct a new regular expression-based resource checker. The options must contain the following required properties: - name - a unique name for this rule - description - a one-line description of what this rule checks for. Example: "Check that URLs in the source also appear in the target" - note - a one-line note that will be printed on screen when the check fails. Example: "The URL {matchString} did not appear in the the target." (Currently, matchString is the only replacement param that is supported.) - sourceLocale - locale (if any) of the source - link - an URL to a document that explains this rule in more detail - severity - severity of the results of this rule. This should be one of "error", "warning", or "suggestion". - regexps - an array of strings that encode regular expressions to look for
